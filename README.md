## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dariamanskaia/YelpCampWDBootcamp
   ```
2. Install NPM packages
   ```sh
   npm install
   ```



<!-- USAGE EXAMPLES -->
## Usage

You can run the web app by running the following command

   ```sh
   nodemon app.js
   ```
